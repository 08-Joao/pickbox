# CSI606-2025-01 - Remoto - Proposta de Trabalho Final
### Discente: Joao Victor Vieira Amora de Figueiredo - 23.1.8019
## Resumo
Será um sistema de armazenamento e compartilhamento de arquivos, levemente inspirado em aplicações como Dropbox, Mediafire ou Mega.io

### 1. Tema
O trabalho tem como tema o desenvolvimento de um aplicação web de armazenamento de arquivos, sendo possível fazer o upload de arquivos, apagar esses arquivos e compartilhar esses arquivos com outros usuários do sistema.

### 2. Escopo
Este projeto terá as seguintes funcionalidades:

- Criar conta e realizar login.
- Realizar logout.
- Fazer upload de arquivos.
- Apagar arquivos.
- Listar arquivos.
- Editar pequenas informações dos arquivos.
- Compartilhar arquivos com outros usuários.

### 3. Restrições
Neste trabalho não serão considerados:

Autenticação avançada (2FA, login por Google ou Microsoft).
Criação e gerenciamento de pastas.
Compartilhamento entre organizações

### 4. Protótipo
Foram elaborados protótipos das principais páginas da aplicação, incluindo:

- Tela de Signin
- Tela de Signup
- Tela de Dashboard
- Tela de My Files
- Tela de Shared With Me
